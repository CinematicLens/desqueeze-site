Read the ring, not the marketing page

Sirui ships multiple squeeze factors even within one focal length family. Before touching sliders, photograph the lens barrel or keep the factory sheet in cloud storage so DITs can match decimals exactly. If your factor lands between 1.5× and 1.8×, use Custom in Web Studio and type the precise value — the preview math updates instantly.

Sirui ingest order

  1. Duplicate camera originals; work on copies labeled SIRUI-<factor> in the filename.
  2. Open Web Studio (or the iOS app if you are still on set).
  3. Enter the squeeze printed on the ring; verify oval bokeh looks round in preview.
  4. Export once for editorial, once for dailies if you need separate LUT stacks — do not bake both into the same MP4.
